## Quick Download
```bash
# Linux
wget https://raw.githubusercontent.com/xizhendu/public-releases/main/qcloud/qcloud-instance-manager-linux
chmod +x qcloud-instance-manager-linux

# macOS
wget https://raw.githubusercontent.com/xizhendu/public-releases/main/qcloud/qcloud-instance-manager-macos
chmod +x qcloud-instance-manager-macos

# Windows (PowerShell)
Invoke-WebRequest -Uri https://raw.githubusercontent.com/xizhendu/public-releases/main/qcloud/qcloud-instance-manager-windows.exe -OutFile qcloud-instance-manager-windows.exe
```
